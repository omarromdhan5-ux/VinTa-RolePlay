# Todo App - قائمة المهام

تطبيق قائمة مهام متقدم مع وظائف Local Storage كاملة

## ✨ الميزات

- ✅ **إضافة وتعديل وحذف المهام** - إدارة كاملة للمهام
- 💾 **Local Storage** - حفظ تلقائي لجميع المهام في الجهاز
- 🔍 **البحث والتصفية** - البحث والتصفية حسب الحالة والأولوية
- 📊 **إحصائيات** - عرض احصائيات المهام
- 🎨 **تصميم جميل** - واجهة حديثة مع Tailwind CSS
- 📥 **استيراد وتصدير** - تحميل واستيراد المهام كـ JSON
- 🏷️ **الفئات والأولويات** - تصنيف المهام حسب الفئة والأولوية
- 📅 **تاريخ الاستحقاق** - تتبع المهام المتأخرة
- 🎯 **ترتيب ذكي** - ترتيب المهام حسب التاريخ والأولوية

## 🚀 البدء

### التثبيت

```bash
cd todo-app
npm install
npm start
```

سيفتح التطبيق على `http://localhost:3000`

### الاستخدام

1. **إضافة مهمة**: اضغط على "إضافة مهمة" وملأ النموذج
2. **تعديل مهمة**: اضغط على أيقونة التعديل
3. **حذف مهمة**: اضغط على أيقونة الحذف
4. **إكمال مهمة**: اضغط على الدائرة بجانب المهمة
5. **البحث**: استخدم شريط البحث للبحث عن مهام
6. **التصفية**: استخدم قائمة التصفية
7. **الترتيب**: ترتيب المهام حسب التاريخ أو الأولوية

## 💾 Local Storage

جميع المهام يتم حفظها تلقائياً في `localStorage`:
- `todos_list` - قائمة المهام
- `todos_settings` - إعدادات التطبيق

## 📤 استيراد/تصدير

- **تحميل (Export)**: احفظ جميع مهامك كملف JSON
- **استيراد (Import)**: استرجع مهامك من ملف JSON
